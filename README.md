# TSQuiz

## Project description

Front-end for a questions from OpenTriviaDB Api.

Live: https://adamjochemczyk.github.io/TSQuiz/

## Technologies

* TypeScript
* Webpack
* SCSS
* HTML, CSS
* TSJest, jest
* REST
